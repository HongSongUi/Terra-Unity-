# Terra-Unity-

## Overview
Unity 기반 보스 전투 중심 액션 게임 프로젝트  
입력 보류 기반 콤보 시스템과 가중치 기반 AI 패턴 선택 구조 구현

## Key Features
- 입력 보류 기반 콤보 처리 (Animation Event 연동)
- 페이즈 기반 보스 AI (Behavior Tree + Weight Set)
- 전투/연출(Camera, Cutscene) 분리 구조

## Code Structure
- Player/ : 플레이어 입력 및 전투 로직
- Boss/ : 보스 전용 패턴 및 페이즈 관리
- BehaviorTree/ : 패턴 선택 액션 노드
- Presentation/ : Camera, Cutscene 제어
